## ❀ Folder Purpose ────୨ৎ────────୨ৎ────

The `t-photos/` folder stores the visual documentation related to the Go!Cheese team. While the rest of the repository focuses on the robot’s mechanical design, sensors, software, testing, and engineering decisions, this folder focuses on the people who worked on the project.

These photos help make the documentation more complete because WRO is not only about the robot. It is also about teamwork, communication, testing, creativity, and the effort behind each improvement.

## ❀ Team Overview ────୨ৎ────────୨ৎ────

We are **Go!Cheese**, a robotics team of two from San Miguelito, Panama, competing in the **WRO Future Engineers 2026** season. Our team is made up of **Caylee Rios** and **Romina Mora**, and each of us contributes to the project in a different but connected way.

<div align="center">

| Team Member | Main Role | Project Focus |
| :--- | :--- | :--- |
| **Caylee Rios** | Documentation and software analysis | README organization, engineering explanations, testing analysis, and project structure. |
| **Romina Mora** | Programming and robot building | Code implementation, robot construction, mechanical changes, and testing support. |

</div>

Together, we work by testing Cheese, observing what happens, discussing the issue, adjusting the robot or code, and documenting the improvement.
