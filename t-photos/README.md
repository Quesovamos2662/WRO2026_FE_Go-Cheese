## ❀ About This Folder ────୨ৎ────────୨ৎ────

The `t-photos/` folder stores the images used to present the Go!Cheese team in the main README. While the rest of the repository focuses on the robot’s mechanics, sensors, software, testing, and engineering decisions, this folder helps show the people who worked on the project.

These photos are important because WRO is not only about the final robot. It is also about teamwork, communication, learning, testing, fixing mistakes, and improving together.

---

## ❀ Go!Cheese Team ────୨ৎ────────୨ৎ────

We are **Go!Cheese**, a robotics team of two from San Miguelito, Panama, competing in the **WRO Future Engineers 2026** season.

<div align="center">

| Team Member | Main Role | Project Focus |
| :--- | :--- | :--- |
| **Caylee Rios** | Documentation and software analysis | README organization, engineering explanations, testing analysis, and project structure. |
| **Romina Mora** | Programming and robot building | Code implementation, robot construction, mechanical changes, and testing support. |

</div>

Together, we work by testing Cheese, observing what happens, discussing the issue, adjusting the robot or code, and documenting the improvement.
